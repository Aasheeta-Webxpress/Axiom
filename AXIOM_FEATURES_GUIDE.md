## üöÄ **Getting Started Quick Guide**

### **Step 1: Setup Backend**
```bash
cd axiomBackend
npm install
npm run dev
```

### **Step 2: Setup Frontend**
```bash
cd axiom
flutter pub get
flutter run
```

### **Step 3: Create Your First App**
1. Register/login to the app
2. Click "New Project" on dashboard
3. Add widgets using drag-and-drop
4. Create APIs for data operations
5. Test with preview mode
6. Export your code
